Salesforce Metadata to Excel Data Dictionary

Instructions:

1. Retrieve your Salesforce metadata using VSCode Salesforce extension (e.g., select "Standard" to generate package.xml and retrieve objects).

2. Zip the retrieved metadata folder (e.g., force-app/main/default/objects/) or select individual .object-meta.xml files.

3. Upload the ZIP file or multiple XML files below.

4. Click "Generate Excel" to create a data dictionary with one sheet per object, listing fields and metadata (e.g., name, type, label, description).

Drag & drop ZIP/XML files here or click to browse.
